사용 방법
=========

1. 이름을 바꿀 PDF 파일들을 이 report_rename_tool 폴더 안에 넣습니다.
2. rename_report_files.bat를 실행합니다.
3. 변경 예정 목록을 확인한 뒤 y를 입력하면 실제 파일명이 변경됩니다.

기본 변경 형식:
국가_상품명(HS코드)_수출시장분석보고서.pdf

예:
001_330749_프래그런스향수_일본_20260520_113644.pdf
-> 일본_프래그런스 향수(330749)_수출시장분석보고서.pdf

주의:
- 이 도구는 기본적으로 report_rename_tool 폴더 안의 PDF 파일만 대상으로 합니다.
- 원본 downloads 폴더를 직접 건드리지 않으려면 PDF를 이 폴더에 복사한 뒤 실행하세요.
- 기존 파일명에서 이미 제거된 띄어쓰기는 복원할 수 없습니다.
- 같은 이름이 이미 있으면 (1), (2)를 붙여 덮어쓰기를 피합니다.
