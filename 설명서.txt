1. install_vm.bat 파일 실행 (처음 1회만 - 프로그램 설치 목적) 
2. run_gui.bat (매번 실행 - 프로그램 실행시)




*참고사항*

<액셀 파일 입력>

기본 템플릿 파일:

   input_template.xlsx

실제 실행용 엑셀은 input_template.xlsx를 복사해서 작성하거나,
GUI에서 원하는 .xlsx 파일을 직접 선택하면 됩니다.

필수 입력값:

- 수출액 규모
- 해당 품목 수출 경험
- HSCODE 6단위 또는 HSCODE 10단위
- 희망진출국가: 수출시장 분석 보고서 생성 선택 시 필요

GUI의 보고서 생성 방식:

- 수출시장 분석 보고서 생성: 희망진출국가를 기준으로 국가별 보고서를 생성합니다. 기본 2개입니다.
- 유망 시장 추천 보고서 생성: 추천 보고서 PDF만 생성합니다.
- 추천 국가로 수출시장 분석보고서 생성 연동: 추천 보고서 생성 후 추천 국가를 추출해 수출시장 분석보고서까지 이어서 생성합니다.

GUI의 분석보고서 수:

- 기본값은 2개입니다.
- 유망 시장 추천 보고서 생성에서 추천 국가로 수출시장 분석보고서 생성 연동을 켰을 때만 표시됩니다.
- 엑셀 희망진출국가와 추천 국가를 합쳐 만들 수출시장 분석보고서 총 개수입니다.

분석 제외 국가는 추천 보고서에서만 사용하며, 여러 국가는 쉼표, 슬래시, 줄바꿈으로 입력할 수 있습니다.


<저장 파일명 커스텀>

기본값은 KOTRA 사이트 기본 파일명으로 저장됩니다.

GUI에서 저장 파일명 커스텀을 켜면 항목과 문자를 조각으로 추가할 수 있습니다.
추가한 조각은 드래그해서 순서를 바꾸거나 x 버튼으로 삭제합니다.

사용 가능 항목:

- 연번: {row_index}
- HS CODE: {hs_code}
- 수출품명: {product_name}
- 희망진출국가: {target_country}
- 분석제외국가: {excluded_countries}
- 보고서생성방식: {report_mode}
- 생성날짜: {date}
- 생성시간: {time}
- 생성일시: {datetime}
- 생성연도: {year}
- 생성월: {month}
- 생성일: {day}
- 생성시: {hour}
- 생성분: {minute}
- 생성초: {second}
- 사이트 기본 파일명: {site_filename}

예시:

{target_country}_{product_name}({hs_code})_수출시장분석보고서

날짜 구분자 예시:

{year}-{month}-{day}_{hour}.{minute}.{second}

저장 예:

베트남_스킨케어(330499)_수출시장분석보고서.pdf


<자동 재시도>

처리 중 실패한 항목은 재시도 대기열에 올라가며 기본 1회에 한하여 다시 시도합니다.
자동 재시도 후에도 실패한 항목만 logs\failed_rows.xlsx에 남습니다.
유망 시장 추천 보고서와 수출시장 분석보고서 연동 작업은 logs\report_tasks.xlsx에 작업별로 기록되며, 재시도 시 이미 완료된 추천 보고서나 국가별 보고서는 건너뛰고 남은 작업부터 이어서 실행합니다.
